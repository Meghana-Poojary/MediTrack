- Node.js and npm

### Clone the Repository

### Install Dependencies
#### For backend
```bash
cd backend
npm install
```
#### For frontend
```bash
cd ../frontend
npm install
```

### Run the Website
#### In backend
```bash
cd backend
nodemon server.js
```
#### In frontend (in a separate terminal)
```bash
cd frontend
npm run dev
```
